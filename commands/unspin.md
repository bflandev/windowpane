Target: $ARGUMENTS

If `$ARGUMENTS` is a URL, fetch it; if a file path, read it; otherwise treat it as the text.
If no target is given, ask what to analyze.

Produce: a numbered `quote → category → plain translation → what it conceals` table using the
9-category evasion taxonomy; a **central evasion** paragraph (the gap between declared and real
aims); and a **fairness note** separating earnest abstraction from bad-faith concealment. Run the
§9 internal-contradiction check but do not force it. **Translate to expose — never rewrite the text
to read "better,"** which would only launder it.

Quote each phrase **once** — in the table. The central-evasion paragraph and fairness note refer
back to rows by number (e.g. "the layoff euphemism, row 1"); they don't re-quote what the table
already holds. The table is the evidence; the prose points at it.
