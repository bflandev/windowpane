# Operators, padding, pretentious diction — lookup & fixes

Covers faults **2 (operators / verbal false limbs)** and **3 (pretentious diction)**. The common
move in both: the writer reaches for a longer, vaguer construction than the meaning needs. The
fix is almost always to **find the buried verb and use it**, or **swap the long word for the
short one** — then run the Rule-6 check.

---

## Operators / verbal false limbs → plain verb

A phrase pretending to be a verb. Cut to the verb inside it.

| Operator phrase | Plain verb |
|-----------------|-----------|
| render inoperative | break, stop |
| militate against | work against, hurt |
| make contact with | meet, reach, call |
| be subjected to | undergo, suffer |
| give rise to | cause |
| give grounds for | justify |
| have the effect of | (just use the verb) |
| play a leading part/role in | lead, drive |
| make itself felt | show, appear |
| exhibit a tendency to | tend to |
| serve the purpose of | (just use the verb) |
| is reflective of | reflects, shows |
| has the capability of | can |
| is in a position to | can |
| take into consideration | consider |
| reach a decision | decide |
| conduct an investigation into | investigate |
| provide assistance to | help |
| make an adjustment to | adjust |
| is indicative of | shows |

**Noun-of-verb over gerund:** *by examination of* → *by examining*; *the implementation of* →
*implementing*.

## Padding connectives — usually cut entirely

*with respect to · with regard to · having regard to · the fact that · in view of the fact that ·
in the interests of · on the hypothesis that · by dint of · in terms of · in order to* (→ *to*) ·
*due to the fact that* (→ *because*) · *for the purpose of* (→ *to*) · *in the event that* (→
*if*) · *at this point in time* (→ *now*) · *in the majority of cases* (→ *usually*) · *a
percentage of* (→ *some*) · *it should be noted that* · *it is important to note that* · *needless
to say.*

## The "-ize" / "de-" manufacture and "not un-"

- **Manufactured verbs:** *operationalize, prioritize, incentivize, leverage (v.), action (v.),
  dialogue (v.), architect (v.).* Prefer the plain verb (*rank, reward, use, do, talk, design*).
- **"not un-" formation** (Rule: kill it). *not unjustifiable → justifiable; not insignificant →
  significant; not unlike → like.* Orwell's antidote sentence: *"A not unblack dog was chasing a
  not unsmall rabbit across a not ungreen field."*

## Pretentious diction → everyday English (Rule 5 / Rule 2)

| Long / Latinate | Plain |
|-----------------|-------|
| utilize | use |
| constitute | make up, form, be |
| facilitate | help, ease |
| endeavour | try |
| ameliorate | improve |
| commence | start, begin |
| terminate | end |
| eliminate | cut, remove |
| demonstrate | show |
| ascertain | find out |
| necessitate | need, require |
| sufficient | enough |
| approximately | about |
| individual (noun) | person |
| in the vicinity of | near, about |
| subsequent to | after |
| prior to | before |
| methodology | method |
| functionality | features, what it does |
| utilization | use |
| remuneration | pay |
| expedite | speed up, hurry |
| cognizant of | aware of |
| predicated on | based on |

**Inflated adjectives** to distrust (Orwell's list + kin): *epoch-making, epic, historic,
unforgettable, triumphant, age-old, inevitable, inexorable, veritable* — and modern: *iconic,
legendary, game-changing, revolutionary, unprecedented, seamless, cutting-edge.* Earn the
adjective with a fact, or cut it.

**Needless foreign phrases** (Rule 5): *deus ex machina, mutatis mutandis, status quo,
weltanschauung, ancien régime, ceteris paribus, inter alia, vis-à-vis.* Use the English unless the
term is genuinely a term of art with no plain equivalent.

---

## Rule-6 guard for this file

These tables are defaults, not commandments. **Keep the long/technical word when:**
- It is a precise term of art whose plain "equivalent" loses meaning (*velocity* ≠ *speed* in
  physics; *passive immunity* ≠ "borrowed protection" in medicine).
- The audience is specialists who share the jargon and the plain version would patronize them.
- The short Saxon word carries a wrong connotation the Latin word avoids.

Always confirm the rewrite *reads better*, not merely *shorter*.
